# My Portfolio
This GitHub repository has been created to host my new portfolio that I have made using HTML, CSS and JavaScript code.

[Link to my portfolio](https://thajeepan-rathiharan.github.io/thajeepan_rathiharan/)

## Software used
- HTML
- CSS
- JavaScript
- Visual Studio Code

## Features
- Fully responsive - can be viewed on desktop and mobile platforms.
- Burger menu navigation bar for mobile devices to navigate between content.
- My CV can be viewed upon clicking the 'View CV' button which will take the user(s) to a public Google Drive folder that has my CV stored in it.
- Coding languages (skills) list - I decided to put this after my 'About Me' section so the user viewing my portfolio can see what skills I currently have before confirming this via the projects I showcase in the following 'Projects' section of my portfolio.
- Users are able to see my projects posted in my 'Projects', which has a short summary of the project, what technology was used to create it and a link to view it.
- Contact page at the end for users to find ways to contact me.
